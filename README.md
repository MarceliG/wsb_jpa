"# wsb_jpa" 

## Run server
```bash
mvn spring-boot:run
```

## Start server
```bash
mvn spring-boot:start
```

## Stop server
```bash
mvn spring-boot:stop
```

## Run tests
```bash
mvn test
```

## Run individual test
```bash
mvn -Dtest=<TestName> test
```
